# SIEDOM — Sistem Evaluasi Dosen oleh Mahasiswa (Web Dashboard)

SIEDOM adalah sistem dashboard berbasis web yang dirancang untuk memvisualisasikan hasil Evaluasi Dosen oleh Mahasiswa (EDOM) secara dinamis. Proyek ini memanfaatkan Google Apps Script sebagai backend, dengan integrasi Google Sheets sebagai sumber data dan tampilan HTML/CSS/JavaScript untuk antarmuka web.

---

## Features
- **Dynamic Dashboard**: Menampilkan data EDOM secara real-time melalui Web App.
- **Google Apps Script Integration**: Backend otomatis terhubung ke Google Sheets.
- **JSON API Output**: Mendukung output data dalam format JSON untuk kebutuhan integrasi.
- **Responsive UI**: Tampilan dashboard kompatibel untuk desktop dan mobile.
- **Ringan & Mudah Dideploy**: Seluruh sistem bisa dipublikasikan langsung melalui Apps Script.
